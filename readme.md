  
<span style="font-family:Arial sans-serif;font-size:16px;">WD HTML Page Import</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">This example explains how HTML pages can be imported with the WLanguage functions.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">The following topics are presented in this example:</span>

<span style="font-family:Arial sans-serif;font-size:14px;">1/ how to import an object found on a Web site</span>

<span style="font-family:Arial sans-serif;font-size:14px;">2/ how to analyze an HTML file</span>

<span style="font-family:Arial sans-serif;font-size:14px;">Summary of the example supplied with WINDEV: </span>

<span style="font-family:Arial sans-serif;font-size:14px;">This example is used to save locally an HTML page found on a Web site. </span>

<span style="font-family:Arial sans-serif;font-size:14px;">This page is analyzed in order to import all its dependencies (images, applets, and so on). This example is not a Web grabber. It can only be used to download the pages one by one.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">The principle used in this example can also be used to retrieve informations from pages whose format is recognized (example: daily retrieval of share values)</span>
